To install the Psychtoolbox follow the steps outlined
in this document.

WINDOWS:
- Download and install Subversion
- Assure you have Microsoft Runtime Libraries
  for MSVC 2010 (or later) installed
- Copy the DownloadPsychtoolbox.m file into the
  directory into which you want to install the
  toolbox
- Launch MATLAB and navigate to the directory
- Execute the following two command
  >> DownloadPsychtoolbox('PATH_TO_DOWNLOAD_TOOLBOX_TO');
- Update the toolbox using
  >> UpdatePsychtoolbox
